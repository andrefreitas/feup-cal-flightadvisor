Detailed Description

A class for describing a program that chooses the best routes according the source and destination airport.

The network of flight paths is represented as a graph and the decisions are made upon graphs algorithms. Remember that an waypoint is a place in a map that a plane must pass. That waypoint can be an airport or not.
